Aplicativo mobile desenvolvido com foco na gestão e operação de uma hamburgueria, oferecendo funcionalidades para organização de pedidos, produtos e fluxo de atendimento.

---

## 📱 Sobre o Projeto

O **G10 Hamburgueria APP** é um aplicativo Android desenvolvido como projeto acadêmico. O objetivo principal é simular um sistema real de gerenciamento, facilitando o controle de pedidos e otimizando a organização operacional de uma hamburgueria.

---

## 🎯 Funcionalidades

- 📦 **Registro de pedidos**: Cadastro rápido e intuitivo.
- 🍔 **Organização de produtos**: Gestão de itens do cardápio.
- 📋 **Controle de fluxo**: Acompanhamento do status do atendimento.
- 📱 **Interface Minimalista**: Focada na experiência do usuário (UX).

---

## 🚀 Tecnologias Utilizadas

O projeto utiliza o que há de mais moderno no ecossistema Android:

- **Linguagem:** [Kotlin](https://kotlinlang.org/)
- **IDE:** Android Studio
- **Build System:** Gradle (KTS)
- **Arquitetura:** Padrão Android (View Binding / Separation of Concerns)

⚙️ Como Executar o Projeto
Para testar o aplicativo localmente, siga os passos abaixo:

Clone o repositório:

```text
git clone [https://github.com/jmpfbr/G10_HamburgueriaAPP.git](https://github.com/jmpfbr/G10_HamburgueriaAPP.git)
```

Abra no Android Studio:
Vá em File > Open e selecione a pasta do projeto.

Sincronize o Gradle:
Aguarde o Android Studio baixar as dependências automaticamente.

Execute:
Selecione um Emulador ou um Dispositivo Físico e clique no botão de "Run" (ícone de play verde).
